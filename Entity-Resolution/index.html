How RAMS Attributes Publications
RAMS counts papers, not people. The foundational principle is simple: if an organisation has any author affiliated with a paper, that organisation gets +1. Different counting models can then be layered on top - but the base attribution must be right first.
CURRENT MODEL Paper-Level Full Count - What RAMS Does Now

It is the paper that counts, not the person. One paper with 100 authors still counts as one paper - regardless of whether 100 of those authors are affiliated with a precinct organisation or only 1 out of 100 is. If any author lists an affiliation, that organisation gets +1. At the precinct level, the same logic applies: if any organisation within a precinct is represented, that precinct gets +1.

The Foundation: Get the Base Attribution Right

Before any counting model is applied, RAMS must correctly answer two questions for every paper: which organisations are genuinely connected to this paper? and which precincts do those organisations belong to?

This is the entity resolution challenge - normalising affiliation strings ("Dept. of Cardiothoracic Surg., RNSH" ‚Üí Royal North Shore Hospital), resolving researcher identities across name variants, and recovering hidden affiliations via StaffLink. If the base attribution is accurate, any counting model can be applied on top. If the base is wrong, no model will produce trustworthy results.

There does not need to be one ultimate counting model. As long as the base attributions are as correct as possible, the choice of how to count - full count, fractional, weighted - can be decided separately for each reporting scenario. The same underlying data supports all of them.

The Principle: Accurate Base, Flexible Counting

There does not need to be one ultimate solution. RAMS stores the complete, validated base attribution - every paper linked to every organisation and precinct that is genuinely connected to it. From that foundation, different counting models can be applied to different reporting scenarios without re-processing the data. Precinct dashboards use full count. Grant evaluations use fractional. Leadership analysis uses author position weighting. The base data is the same - only the lens changes.

Attribution Model Options
Five models for handling multi-affiliation attribution, each suited to different analytical purposes. RAMS can apply any of these - the choice depends on the question being asked.
1. Full Count (Whole Count) Current Default
Used by: Times Higher Education, most institutional reports
Every listed affiliation receives full credit (1.0) for the publication. A paper with 3 affiliations generates a count of 1 for each organisation. This inflates system-wide totals but reflects each institution's genuine involvement.
Prof. Smith's Paper (3 listed affiliations) USyd = 1.0  |  RNSH = 1.0  |  Kolling = 1.0  |  NSLHD = 0.0
StrengthsNo organisation loses credit. Simple to explain. Standard in bibliometrics.
LimitationsDouble-counts at system level. Can't assess fair "share" of output.
2. Fractional Count (Pro Rata) Recommended for Grants
Used by: Leiden Rankings, CWTS, Norwegian funding model
Publication credit is divided equally among all listed affiliations. With 2 affiliations, each gets 0.5. This ensures system-wide total = 1.0 per paper, preventing inflation. Best for evaluating return on investment.
Prof. Smith's Paper (3 listed affiliations) USyd = 0.33  |  RNSH = 0.33  |  Kolling = 0.33  |  NSLHD = 0.0
StrengthsNo double-counting. Fair distribution. Gold standard for funding evaluation.
LimitationsPenalises collaborative institutions. Hospitals still invisible if not listed.
3. First-Listed (Primary Affiliation)
Used by: Some national assessments, simplistic reporting
Only the first-listed affiliation receives credit. Reflects the well-known cascade where researchers list university first, then MRI, then hospital. Quick to implement but systematically disadvantages hospitals and MRIs.
Prof. Smith's Paper (USyd listed first) USyd = 1.0  |  RNSH = 0.0  |  Kolling = 0.0  |  NSLHD = 0.0
StrengthsNo inflation. Very simple. No ambiguity.
LimitationsDeeply unfair. MRIs and hospitals systematically lose. Reinforces university bias.
4. Weighted Attribution (Configurable)
Used by: Custom institutional analyses
Assigns configurable weights based on affiliation order, type, or other signals. For example: 50% to university, 30% to MRI, 20% to hospital. Can incorporate StaffLink data to include unlisted affiliations with a recovery weight.
Prof. Smith's Paper (with StaffLink recovery) USyd = 0.40  |  RNSH = 0.25  |  Kolling = 0.20  |  NSLHD = 0.15
StrengthsMost nuanced. Configurable per use case. Can account for hierarchy and recovery.
LimitationsComplex to explain. Weight choices are subjective. Requires stakeholder agreement.
5. StaffLink-Enhanced Attribution RAMS Unique Capability
Not available in any commercial bibliometric platform (Scopus, Dimensions, OpenAlex)
This is RAMS' competitive differentiator. By linking StaffLink IDs (from REGIS user data or Entra ID) to cost centres and employing organisations, RAMS can recover affiliations that researchers never listed on their publications. This is the only method that addresses the systematic hospital undercount problem.

StaffLink enhancement can be combined with any of the above models. For example, "Full Count + StaffLink" gives every organisation (including recovered hospitals) a count of 1.0, while "Fractional + StaffLink" divides the publication equally among all affiliations including the recovered one.

Recommended Approach

RAMS should implement multiple attribution models simultaneously rather than choosing one. The platform can store the full affiliation data and apply the appropriate model at the point of reporting. This means precinct dashboards use Full Count + StaffLink for comprehensive mapping, while grant evaluation reports use Fractional + StaffLink for fair distribution. The underlying data is the same - only the counting logic changes. This is a configuration choice, not a data architecture change.

How StaffLink Addresses Key Attribution Questions
Q1: How do we correctly attribute publications to hospitals even when not acknowledged?

StaffLink provides the employment relationship independent of publication metadata. If a researcher is employed by Royal Prince Alfred Hospital (confirmed by StaffLink), then all their publications are linked to RPAH/Sydney LHD regardless of what they wrote on the paper. The publication affiliation data shows where they said they work; StaffLink shows where they actually work.

Q2: How do we find publications where OHMR funding wasn't acknowledged?

RAMS links REGIS grant records to researcher profiles. If Dr. X received an OHMR-administered grant (recorded in REGIS), and Dr. X published a paper (found via PubMed/Crossref), RAMS can flag papers published during or after the grant period that don't include an OHMR acknowledgement. This is automated - no need to ask researchers directly.

Q3: How do we use StaffLink for fair pro-rata attribution including hospitals and MRIs?

By recovering the LHD affiliation via StaffLink, RAMS increases the denominator in fractional counting. Instead of splitting a publication three ways between university, hospital, and MRI (ignoring the LHD), it becomes four ways - 0.25 each. At the precinct level, since RNSH, Kolling, and NSLHD all map to Sydney North, that precinct's share becomes 0.75 versus SIPFHER's 0.25. The total still sums to 1.0, but the distribution is fairer.

The Counting Problem
When a publication has authors from multiple organisations, how should each organisation be credited? The bibliometric literature has proposed at least 32 distinct counting methods since 1981. The choice of method can change the result of an analysis by 2x or more.

The same paper produces a system-wide total of 16 credits under full counting or 1 credit under fractional counting. Neither is "wrong." They answer different questions. Full counting answers "which organisations were involved in this research?" Fractional counting answers "how should credit for this one paper be distributed fairly?" The appropriate method depends on the purpose of the analysis, which is why the literature contains so many alternatives.

The Landscape: Eight Major Counting Families
The bibliometric literature has documented at least 32 distinct counting methods (Gauffriau 2021, Quantitative Science Studies). Most fall into a few broad families. The eight most relevant to institutional and precinct-level attribution are listed below, along with their mathematical properties, who uses them, and their trade-offs.
1. Full Count (Whole Count) Current RAMS Default
Used by: UK REF, Times Higher Education, QS Rankings, SciVal, InCites
Every organisation with at least one affiliated author receives a full credit of 1.0 for the publication. A 15-author paper across 16 organisations generates a count of 1 for each. System-wide totals exceed the number of unique papers.
Formula Credit per org = 1.0 (if any author affiliated)
StrengthsSimple. No organisation loses credit. Standard in institutional benchmarking. Preserves the base unit (1 paper = 1 paper).
LimitationsInflates system-wide totals. A paper with 16 orgs creates 16 credits. Cannot be summed across orgs without deduplication.
2. Fractional Count (Complete-normalised) Leiden Ranking Default
Used by: CWTS Leiden Ranking, Norwegian funding model (original), Aksnes et al. 2012
Credit is divided equally among all unique organisations. With 4 orgs, each gets 0.25. System-wide totals always equal the number of unique papers. Recommended by Perianes-Rodriguez, Waltman & van Eck (2016) as preferable for most purposes.
Formula Credit per org = 1 / N (where N = number of unique affiliated organisations)
StrengthsNo inflation. Totals are additive. Gold standard for funding evaluation. Used by the world's most methodologically rigorous ranking (Leiden).
LimitationsFragments the base unit (a Nature paper becomes "0.06 of a paper"). Can penalise highly collaborative institutions. Politically difficult to explain.
3. Straight Count (First-listed Only)
Used by: Some national assessments, Lotka's original law (1926)
Only the first-listed author's primary affiliation receives credit. All other organisations receive zero. Lotka himself noted this was a simplification: publications "should perhaps be considered separately since they are not the product of one person unassisted."
Formula Credit: 1.0 for first-listed org, 0.0 for all others
StrengthsNo inflation. No ambiguity. Simple to implement.
LimitationsDeeply unfair. Systematically disadvantages hospitals and MRIs (researchers typically list university first). Not recommended for any modern evaluation.
4. Harmonic Count Best Empirical Fit
Proposed by: Hodge & Greenberg 1981, validated by Hagen 2010
Credit decreases by a harmonic progression based on author position: the 1st author gets twice as much as the 2nd, the 2nd gets 1.5x the 3rd, and so on. Validated against empirical data from medicine, psychology and chemistry, where it fits perceived credit allocation better than any other fixed-type method (Hagen 2010, Scientometrics).
Formula Credit for author i = (1/i) / H(N), where H(N) is the N-th harmonic number. For 10 authors: 1st = 34.1%, 2nd = 17.1%, 3rd = 11.4%, ... last = 3.4%
StrengthsBest validated against how researchers actually perceive credit. Respects author ordering conventions in biomedicine. Ethical: all authors get non-zero credit.
LimitationsAssumes author ordering is meaningful (not true in all fields, e.g. alphabetical ordering in mathematics/economics). More complex to explain to non-technical audiences.
5. Arithmetic Count (Proportional)
Proposed by: Van Hooydonk 1997
Credit decreases linearly by author position. The 1st author of a 5-author paper gets 5/15, the 2nd gets 4/15, and so on. The ratio of first-to-last author credit equals the total number of authors.
Formula Credit for author i = (N - i + 1) / sum(1..N). For 5 authors: 1st = 33%, 2nd = 27%, 3rd = 20%, 4th = 13%, 5th = 7%
StrengthsIntuitive linear decline. All authors receive credit. Total sums to 1.0.
LimitationsIntermediate fit to empirical data (worse than harmonic). Over-credits first author relative to perceived norms in some fields.
6. Geometric Count
Proposed by: Egghe et al. 2000, extended by Liu & Fang 2023
Credit decreases by a constant ratio between consecutive authors. The "adaptive geometric" variant (Liu & Fang 2023, Journal of Informetrics) adjusts the ratio based on team size, achieving the best overall fit of any fixed-type method when tested against the combined empirical dataset from medicine, psychology and chemistry.
Formula Credit for author i = r^(i-1) / sum(r^0..r^(N-1)), where r is the common ratio. Adaptive variant sets r so that first/last ratio = N.
StrengthsMathematically elegant. Bounded total credit even with first-author normalisation. Strong empirical fit in adaptive variant.
LimitationsMultiple parameterisations exist. The "golden ratio" variant (Berker 2018) challenges the assumption that total credit must equal 1.0.
7. Modified Fractional Counting (MFC) Norwegian National Model
Proposed by: Sivertsen, Rousseau & Zhang 2019. Used by: Norway (since 2018), Aalborg University Denmark
A parametric family that sits on a continuum between full counting (k → ∞) and complete fractional counting (k = 1). Norway uses this for allocating national research funding across all universities and hospitals. The key property: MFC discourages neither collaboration (as pure fractional would) nor hyperauthorship (as pure full count would). Proven to be a weighted geometric average of the two extremes (Rousseau et al. 2025, Quantitative Science Studies).
Formula For an author in an N-author paper: credit = N^(k-1) / N^k = 1/N for k=1, approaches 1 for large k. Norway's implementation uses institutional-level MFC for funding allocation.
StrengthsFormally proven to balance between full and fractional. Used in real national funding systems. Tunable via parameter k. Does not penalise collaboration as heavily as pure fractional.
LimitationsChoice of k is ultimately a policy decision. Possible perverse incentives for mutual co-authorship between institutions (Donner 2024, Journal of Informetrics).
8. Funding Acknowledgement Matching Best for Grant Programs
Infrastructure: Crossref Open Funder Registry (35,000+ funders), Grant Linking System (125,000+ grants), Europe PMC Grant Finder
Instead of asking "which organisations were involved?", this method asks "which funding sources supported this work?" Credit is attributed to the grant or funding body acknowledged in the paper. Crossref's Open Funder Registry provides standardised funder IDs, and the Grant Linking System (launched 2019) assigns persistent DOIs to individual grants. Europe PMC provides searchable grant-to-publication linkages. This avoids the multi-affiliation inflation problem entirely because the attribution unit is the funding source, not the organisation.
Pipeline Parse funding/acknowledgement section of the paper. Match funder names to Crossref Funder Registry IDs. Match grant numbers to grant records. One paper per acknowledged grant.
StrengthsDirectly answers "what did this investment produce?" No multi-affiliation inflation. Defensible and self-explanatory. Growing infrastructure support from Crossref, Europe PMC, Wellcome, and 35+ other funders globally.
LimitationsNot all papers acknowledge their funding sources. Scopus uses NLP/ML to extract acknowledgements, but coverage is incomplete. Only works for publications (not patents, datasets, etc. unless separately tracked).

KEY DISTINCTION Two Different Questions Require Different Approaches
Question A: "What does this precinct produce?"

This is an affiliation-based question. The answer depends on which counting method is chosen: full count, fractional, harmonic, MFC, etc. There is no single "correct" answer, only different lenses for different purposes. Standard bibliometric tools (Leiden, SciVal, InCites) all support multiple methods.

Methods: Full, Fractional, Harmonic, Arithmetic, Geometric, MFC, Straight
Question B: "What did this grant investment produce?"

This is a funding-based question. The answer is tied to the funding source, not the author affiliations. One paper crediting one grant = one paper for that grant. The multi-affiliation inflation problem does not arise. Crossref and Europe PMC provide infrastructure for this approach.

Methods: Funding acknowledgement matching, Grant-researcher linkage, Crossref GLS

How This Applies to RAMS

RAMS stores the complete, validated data at the most granular level: every author-organisation-publication triple, every funding acknowledgement, every grant-researcher link recovered from REGIS. From this single data layer, any counting method can be applied at the point of reporting without re-processing the underlying data. A precinct dashboard can use full counting while a grant evaluation report uses funding acknowledgement matching. The choice of counting method is a reporting configuration, not a data architecture decision.

On harmonic counting: Hagen (2010) showed that harmonic credit allocation fits empirical data from medicine 18x better than fractional counting and 2x better than arithmetic counting. The key advantage for biomedical contexts is that it respects the convention of first author (did the work) and last author (supervised/funded), which is exactly the convention in NSW Health research. The limitation is that it requires author ordering to be meaningful, which is true in biomedicine but not in all fields.

What the inflation column means: Under full counting, the three precinct columns add to 1,110, but there are only 500 unique papers. This is not an error. It reflects the reality that many papers are connected to more than one precinct. But if the three numbers are summed into a single "NSW total" without deduplication, the result overstates the unique output by 2.2x. The practical safeguard is straightforward: show per-precinct columns (full count), but always report totals as the deduplicated unique count.
GRANT ATTRIBUTION Funding Acknowledgement Matching: How It Works

Three scenarios showing how funding-based attribution handles common situations. The key advantage: the multi-affiliation counting debate is avoided entirely.

Scenario A: Paper explicitly acknowledges the grant
The paper states "Funded by NHMRC Grant APP1234567." RAMS matches this grant ID to the program record. Result: 1 paper credited to the program. Clean, defensible, no ambiguity. This is the same approach used by Wellcome Trust and Europe PMC for programmatic grantee reporting.
Scenario B: Paper does not acknowledge funding
Dr. Smith received a Program X grant (recorded in REGIS). She published 3 papers during the grant period but did not acknowledge the funding. RAMS links Dr. Smith to her publications via entity resolution. Result: 3 papers flagged as probable program outputs. Reported separately with a lower confidence tier so that explicitly acknowledged and inferred outputs can be distinguished.
Scenario C: Paper acknowledges multiple grants from the same program
The paper lists "NHMRC APP1234567 and NHMRC APP7654321," both from Program X. RAMS counts this as 1 paper for the program (not 2), because deduplication occurs at the program level. At the individual grant level, both grants receive credit. This mirrors the Crossref Grant Linking System design, where each grant has its own DOI but program-level reporting aggregates and deduplicates.

Which Method for Which Question?
Question Recommended Method Rationale
"What research is connected to this precinct?" Full Count + StaffLink Comprehensive picture of all research activity, regardless of credit distribution
"What is this precinct's fair share of output?" Fractional or MFC Additive totals, no inflation. Leiden Ranking standard or Norwegian parametric balance.
"What did this grant program produce?" Funding Acknowledgement Match Directly tied to investment. No multi-affiliation inflation. Self-explanatory to auditors.
"Who is leading the research?" Harmonic or Author-Position Respects biomedical authorship conventions. Distinguishes PIs from collaborators.
"How do we compare internationally?" Full Count (without StaffLink) Aligns with THE, QS, and standard institutional benchmarks for comparability
"Which papers did funded researchers never acknowledge?" Grant-Researcher Linkage via REGIS Cross-reference REGIS grant recipients with their publications during the grant period

The Principle: Same Data, Different Lens

The choice of counting method is a reporting decision, not a data decision. RAMS ingests and validates all author-organisation-publication-funding relationships once, then applies whichever method is appropriate for the audience and purpose. Adding a new counting method in the future requires only a new query configuration, not a new data pipeline. The bibliometric literature has spent 40+ years debating which method is "best." The answer is that it depends on the question being asked.
